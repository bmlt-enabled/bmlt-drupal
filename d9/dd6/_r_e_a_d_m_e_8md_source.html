1 DESCRIPTION
2 -----------
3 
4 The Basic Meeting List Toolbox (BMLT) is a powerful, database-driven system for tracking NA meetings.
5 It is NOT an official product of [NA](http://na.org). Rather, it is a project designed and implemented by
6 NA members, and meant to be used by official NA Service bodies.
7 
8 This project implements a "standalone satellite." It does not have to be integrated into a content management
9 system.
10 
11 REQUIREMENTS
12 ------------
13 
14 The project requires a functioning [BMLT root server](http://bmlt.app/root-server).
15 It does not implement a root server, but connects to an existing one.
16 It requires a Web server capable of executing PHP 5.0 or above.
17 
18 This class uses the BMLT Satellite Base Class, which is available on [GitHub, here](https://github.com/bmlt-enabled/bmlt-satellite-base-class/).
19 
20 INSTALLATION
21 ------------
22 
23 You install the uncompressed 'bmlt' directory (from whichever zipped file represents your Drupal instance) in /sites/all/modules/ (You may need to create the "modules"
24 directory).
25 Enable the module in the usual manner.
26 This module operates through the use of text filters/input formats. You need to set up a text filter that uses the
27 "Add a BMLT instance inline in text" filter.
