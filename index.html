DESCRIPTION

The Basic Meeting List Toolbox (BMLT) is a powerful, database-driven system for tracking NA meetings. It is NOT an official product of NA. Rather, it is a project designed and implemented by NA members, and meant to be used by official NA Service bodies.

This project implements a "standalone satellite." It does not have to be integrated into a content management system.

REQUIREMENTS

The project requires a functioning BMLT root server. It does not implement a root server, but connects to an existing one. It requires a Web server capable of executing PHP 5.0 or above.

This class uses the BMLT Satellite Base Class, which is available on GitHub, here.

INSTALLATION

You install the uncompressed 'bmlt' directory (from whichever zipped file represents your Drupal instance) in /sites/all/modules/ (You may need to create the "modules" directory). Enable the module in the usual manner. This module operates through the use of text filters/input formats. You need to set up a text filter that uses the "Add a BMLT instance inline in text" filter.
